Getting Started


Prerequisites

To Run this project, go to the MIPS repository.

Before running this project, make sure you have following dependencies -
-GNU C/C++ Compiler
-Python 3.7

Now, using pip install command, include following dependencies
-Tkinter
-Subprocess

Running

To run the project, perform following steps -

Take the all the required files and put them in the folder as mentioned.

Required files are -

mips simulator.py (Code folder)
mips cache simulator.exe (Files folder)
gui.png (Images folder)
mips pipeline simulator.exe (Files folder)

Run the mips simualtor.py file to get the tkinter window as shown below:



Now, click on the respective button to run the desired simulator as below.
